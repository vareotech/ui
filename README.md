# @vareotech/ui

Base do design system da vareo.tech construída com React, Vite, Tailwind, primitives do shadcn/Radix e Storybook.

## Stack

- React
- Vite
- Tailwind CSS
- shadcn/ui como fundação de primitives
- Storybook para documentação e visualização

## Estrutura

```text
src/
  components/ui/
  stories/
    docs/
    foundations/
  templates/
  tokens/
  lib/
```

## Scripts

- `npm run dev`: ambiente local
- `npm run build`: build de produção
- `npm run lint`: validação estática
- `npm run storybook`: Storybook local
- `npm run build-storybook`: build estático da documentação

## Direção visual

- dark-first
- minimalista e profissional
- contraste alto sem agressividade
- base neutra com destaque moderado
- consistente entre dashboards, marketing e experiências mobile
