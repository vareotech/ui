# Checkbox

Checkbox supports multiple selection and explicit opt-in states.

---

## When to use

- Use when users can select zero, one, or many options.
- Use for explicit consent or optional settings.

## Variants

| Variant | Purpose |
| --- | --- |
| Default | Standard binary selection. |
| Disabled | Shows unavailable selection. |

## Do's and Don'ts

### Do's

- Keep labels short and specific.
- Use checkbox only for independent choices.

### Don'ts

- Don't use checkbox when only one option can be chosen.

## Usage

```jsx
<Checkbox aria-label="Enable reports" defaultChecked />
```
