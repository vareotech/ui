# Dialog

Dialog creates a focused layer for high-priority decisions.

---

## Table of Contents

## When to use

- Use when users must confirm or review a focused task.

## Variants

| Variant | Purpose |
| --- | --- |
| Default | Modal decision layer with trapped focus. |

## Do's and Don'ts

### Do's

- Keep title, description, and actions explicit.

### Don'ts

- Don't use dialog for low-priority passive information.

## Usage

```jsx
<Dialog>
  <DialogTrigger>Delete</DialogTrigger>
</Dialog>
```
