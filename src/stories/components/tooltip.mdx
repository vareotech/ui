# Tooltip

Tooltip provides short contextual help close to the trigger.

---

## When to use

- Use for brief clarifications that should not occupy layout space.

## Variants

| Variant | Purpose |
| --- | --- |
| Default | Hover/focus contextual hint. |

## Do's and Don'ts

### Do's

- Keep content brief.

### Don'ts

- Don't place essential instructions only inside tooltip.

## Usage

```jsx
<TooltipContent>Contextual help</TooltipContent>
```
