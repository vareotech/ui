# Sheet

Sheet reveals supplemental detail while keeping the main context visible.

## When to use

- Use when users still need the underlying page context.

## Variants

| Variant | Purpose |
| --- | --- |
| Right sheet | Supplemental task or detail panel. |

## Do's and Don'ts

### Do's

- Keep the content task-oriented and contextual.

### Don'ts

- Don't use sheet when full modal isolation is required.

## Usage

```jsx
<Sheet>
  <SheetTrigger>Open details</SheetTrigger>
</Sheet>
```
