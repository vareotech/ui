# Radio Group

Radio Group supports a single choice inside a defined set of options.

---

## Table of Contents

1. [When to use](#when-to-use)

---

## When to use

- Use when exactly one option should remain selected.
- Use for mutually exclusive plans, modes, or filters.

## Variants

| Variant | Purpose |
| --- | --- |
| Default | Single selection among peers. |

## Do's and Don'ts

### Do's

- Keep all options visible together.
- Order choices logically.

### Don'ts

- Don't use radio group for multi-select behavior.

## Usage

```jsx
<RadioGroup defaultValue="growth" />
```
