# Dropdown Menu

Dropdown Menu exposes secondary actions without increasing layout noise.

## When to use

- Use for compact contextual actions.

## Variants

| Variant | Purpose |
| --- | --- |
| Default | Opens an action list near the trigger. |

## Do's and Don'ts

### Do's

- Keep labels task-oriented and scannable.

### Don'ts

- Don't hide the primary action only inside a dropdown.

## Usage

```jsx
<DropdownMenu>
  <DropdownMenuTrigger>Actions</DropdownMenuTrigger>
</DropdownMenu>
```
