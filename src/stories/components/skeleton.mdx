# Skeleton

Skeleton hints the final layout before real content is ready.

## When to use

- Use to reduce layout jumps during loading.

## Variants

| Variant | Purpose |
| --- | --- |
| Default | Neutral placeholder block. |

## Do's and Don'ts

### Do's

- Match skeleton shape to the incoming layout.

### Don'ts

- Don't combine skeleton with already-loaded final content in the same slot.

## Usage

```jsx
<Skeleton className="h-10 w-full" />
```
