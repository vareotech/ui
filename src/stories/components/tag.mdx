# Tag

Tag groups small attributes and removable filters in a compact chip format.

---

1. [When to use](#when-to-use)

---

## When to use

- Use for categories, selected filters, or compact labels.

## Variants

| Variant | Purpose |
| --- | --- |
| Default | Standard chip label. |
| Removable | Displays removal affordance. |

## Do's and Don'ts

### Do's

- Use removable tags when users can clear a filter.

### Don'ts

- Don't overload tags with full sentence content.

## Usage

```jsx
<Tag removable>Release 1.2</Tag>
```
