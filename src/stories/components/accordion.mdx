# Accordion

Accordion expands and collapses content progressively inside a vertical flow.

## When to use

- Use for optional detail and progressive disclosure.

## Variants

| Variant | Purpose |
| --- | --- |
| Single | Opens one item at a time. |

## Do's and Don'ts

### Do's

- Use clear trigger labels.

### Don'ts

- Don't hide essential primary content by default.

## Usage

```jsx
<Accordion type="single" collapsible />
```
